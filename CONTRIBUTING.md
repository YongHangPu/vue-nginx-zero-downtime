# Contributing

感谢你为 `vue-nginx-zero-downtime` 做贡献。

## 贡献范围

比较欢迎以下类型的改进：

- 文档修正与翻译
- Vue / Vite 接入示例
- Nginx 配置模板增强
- 发布脚本兼容性增强
- 组件可访问性与交互体验优化

## 本地开发

```bash
pnpm install
pnpm dev
```

构建检查：

```bash
pnpm build
```

## 提交参考

- 尽量保持改动聚焦，避免顺手重构无关代码
- 如果方便的话，可以补充文档或示例，说明为什么要这样改
- 如果改了模板文件，可以同步看看 README 是否需要更新
- 如果改了演示页，建议确认 `pnpm build` 仍然通过

## Issue / PR 参考

- Bug：可以说明复现步骤、预期行为、实际行为
- Feature：可以说明使用场景、解决的问题、是否会影响现有模板
- PR：如果方便，附上截图、运行方式或接入说明会更有帮助
